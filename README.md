# boot-typedclojure

```
[org.typedclojure/boot-typedclojure "0.4.0"]
```

Plugin for boot to use Typed Clojure.

## Usage

In boot, add the following to your build file.

```
(require '[typedclojure.boot :refer :all])`
```
